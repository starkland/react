## How to use

[![Build Status](https://travis-ci.org/starkland/react.svg?branch=master)](https://travis-ci.org/starkland/react)

Install dependencies:

```bash
$ npm install
```

Start the application:

```bash
$ npm start
```

Running the tests:

```bash
$ npm test
```
